Api for QTI

POST upload_package
packages.json
json item list of the folders in packages
kennisnet-1/items.json
Items in a package folder
/kennisnet-1/items/ITEM002.xml
The xml of the qti item with corresponding name
kennisnet-1/items/ITEM002.xml?noresponsexml=true
The xml of with the correct answer and responseprocessing tag and children stripped
qti.xsd
Getting the XSD of our light player to use schema, for use in code completion
POST saveeditxml/kennisnet-1/M6.LO6_3.xml
Saves the editted XML back into the item, for use in edit functionality
POST response/kennisnet-1/M6.LO6_3.xml
JSON response saving of one interaction of one item Responses come in like : 
{
    item: "ITM001",
    interaction: {
        responseIdentifier: "A",
        responses: []
    }
}

{
    item: "ITM001",
    interaction: {
        responseIdentifier: "B",
        responses: []
    }
}
GET response/kennisnet-1/01109.xml
Will be delivered back in an array of interactions, which in itself can be set back on the items
{
    item: "ITM001",
    interactions: [
        {
        responseIdentifier: "A",
        responses: []
        }, {
        responseIdentifier: "B",
        responses: []
        }
    ]
}
GET itemresponse/reset
Resets the data on the server
